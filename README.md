<!-- GETTING STARTED -->
## Getting Started

#### Prerequisites

* Node



#### Installation


1. Clone the repo
   ```sh
   gh repo clone selaguardia/Workout-Tracker
   ```
##### Server Side:
1. Install NPM packages
   ```sh
   npm install
   ```
2. Run backend server
    ```sh
      npm run server
    ```
##### Client Side: 
1. Install NPM packages
   ```sh
   npm install
   ```
2. Run React frontent 
    ```sh
      npm start
    ```
